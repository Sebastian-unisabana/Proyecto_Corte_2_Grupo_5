## Descripción del Proyecto

Sistema de gestión para la Cafetería U. Sabana, desarrollado con arquitectura
orientada a objetos (POO) e integrado con una base de datos SQLite.

Esta segunda entrega escala la solución implementando jerarquías de clases,
encapsulamiento, relaciones entre tablas mediante llaves primarias y foráneas,
y operaciones CRUD.

---

## Arquitectura

- Lenguaje: Python
- Base de datos: SQLite (archivo .db persistente)
- Paradigma: Programación Orientada a Objetos (POO)
- Entorno: Jupyter Notebook

---

## Estructura del Repositorio

- `design/` — Diagrama de Clases (POO) y Modelo Entidad-Relación (ERD)
- `app_escalada_grupo_#.ipynb` — Código fuente documentado y funcional
